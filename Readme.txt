

Before Compilation go to the CMakeLists.txt in the sourcecode

1) There you need to change the path for Qt5 paths, where required.
2) If your opencv is build locally, then you need to provide the path of opencv library
3) Else it will do fine.



##########################COMPILATION##############################################




1) Create a build directry.
2) Go to terminal and go to the build directry.
3) type-> cmake ../sourcecode (in terminal)
4) type-> make
5) Anveshak executable within build directry is ready to use.
